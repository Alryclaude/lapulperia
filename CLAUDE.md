# La Pulperia - Project Context

## Overview
Marketplace platform for Honduran pulperias (corner stores)

## Tech Stack
- **Frontend**: React + Vite + TailwindCSS + Leaflet
- **Backend**: Node.js + Express + Prisma
- **Database**: PostgreSQL
- **Storage**: Cloudinary
- **Auth**: Firebase
- **Hosting**: Render (free tier)

## Project Structure
- `/frontend` - React SPA
- `/backend` - Express API server

## Key Commands
```bash
# Frontend
cd frontend && npm run dev

# Backend
cd backend && npm run dev

# Build
npm run build
```

## Conventions
- Spanish UI, English code
- Use TailwindCSS for styling
- Prisma for database operations
- Mobile-first responsive design

## Current Status
- 15+ pages complete
- Interactive map with Leaflet
- PWA with push notifications
- Gamified dashboard for pulperias
